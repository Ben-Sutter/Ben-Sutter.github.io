Assume v is a vector in R^4. You wish to compute a vector w such that the i'th entry of w,

denoted by w_i, is the sum of v_(i-1), v_i, v_(i+1) For example, w_2 is the sum of v_1, v_2, v_3.

For elements at the endpoints of w, you will only account for two elements in your sum.

For example w_0 (assuming zero based indexing) is the sum of v_0 and v_1.

Construct a matrix A such that Av = w. Fill in the entries of A below.

A =

1 1 0 0

1 1 1 0

0 1 1 1

0 0 1 1

For Monte Carlo convergence questions, if given n and estimated error:

First find Z by using the given error and n and solve: error = Z / sqrt(n)

Then use that newly found Z with the same equation and the new n to find the new approx error.

Sometimes they might switch this around and make you find n given a different goal error

For this you must make sure to remember to square the calculated number to get n.

---------------------------------------------------

When given vectors/ matrices and told to calculate a norm/norm bound for another vector/matrix,

Add/subtract the vector(s)/matrix(matrices) together to get the one you are asked to calculate

Count the number of additions/subtractions and multiply the norm by that amount of operations done

Mx = c for a given vector c

As Assistant to the Regional Manager, Dwight wants to test your knowledge and asks you to estimate the

time t that takes to factorize and efficiently solve the system Ax = b, where A is a square matrix with

dimension 450, given 40 different right-hand side b vectors. Dwight says you should not solve the

problem, but instead obtain your estimate using the information from Michael's demonstration. Provide

your time estimate t in the answer box below.

(1350/450)^3 = 243/x -> x = 9 <----- factor time

(1350/450)^2 = 81/x -> x = 9 <----- triangle time

factor time + (#of b vectors * triangle time) = total time
